# VD-001 AI 厂商配置数据结构实现细节

## 实现内容

### 1. TypeScript 类型定义 (src/types/index.ts)

```typescript
// AI 厂商配置接口
export interface AIConfig {
  provider: AIProvider; // AI 厂商标识
  apiKey?: string; // API 密钥
  baseUrl: string; // API 基础 URL
  model: string; // 使用的模型
  enabled: boolean; // 是否启用
  name?: string; // 配置名称（可选）
  createdAt?: number; // 创建时间
  updatedAt?: number; // 更新时间
}

// AI 厂商元数据接口
export interface AIProviderMeta {
  id: AIProvider; // 厂商标识
  name: string; // 显示名称
  docUrl?: string; // 文档链接
  apiKeyUrl?: string; // 获取密钥链接
  defaultBaseUrl: string; // 默认 API URL
  supportedModels: string[]; // 支持的模型列表
  defaultModel: string; // 默认模型
  supportsStreaming: boolean; // 支持流式输出
  supportsVision?: boolean; // 支持视觉功能
}
```

### 2. Rust 模型定义 (src-tauri/src/models/mod.rs)

```rust
/// AI Provider configuration
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct AIProviderConfig {
    pub provider: String,        // AI 厂商标识
    pub api_key: Option<String>, // API 密钥
    pub base_url: Option<String>, // API 基础 URL
    pub model: String,           // 使用的模型
    pub enabled: bool,           // 是否启用
    pub name: Option<String>,    // 配置名称
}

/// AI Provider metadata information
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct AIProviderMeta {
    pub id: String,                    // 厂商标识
    pub name: String,                  // 显示名称
    pub doc_url: Option<String>,       // 文档链接
    pub api_key_url: Option<String>,   // 获取密钥链接
    pub default_base_url: String,      // 默认 API URL
    pub supported_models: Vec<String>, // 支持的模型列表
    pub default_model: String,         // 默认模型
    pub supports_streaming: bool,      // 支持流式输出
    pub supports_vision: Option<bool>, // 支持视觉功能
}
```

### 3. Tauri 命令 (src-tauri/src/commands/ai.rs)

```rust
/// 获取所有 AI 厂商的元数据信息
#[tauri::command]
pub fn get_ai_providers() -> Vec<AIProviderMeta> {
    AIProviderMeta::get_all_providers()
}

/// 获取指定 AI 厂商的元数据信息
#[tauri::command]
pub fn get_ai_provider(provider_id: String) -> Option<AIProviderMeta> {
    AIProviderMeta::get_provider(&provider_id)
}
```

### 调用后端 API

```typescript
import { invoke } from '@tauri-apps/api/core';

// 获取所有 AI 厂商元数据
const providers = await invoke<AIProviderMeta[]>('get_ai_providers');

// 获取当前已保存的配置
const configs = await invoke<AIConfig[]>('get_ai_configs');

// 保存配置
await invoke('save_ai_config', {
  config: {
    provider: 'openai',
    apiKey: 'sk-...',
    baseUrl: 'https://api.openai.com/v1',
    model: 'gpt-4o-mini',
    enabled: true,
  },
});

// 删除配置
await invoke('remove_ai_config', { provider: 'openai' });

// 验证 API 密钥
const isValid = await invoke<boolean>('validate_ai_key', {
  provider: 'openai',
  api_key: 'sk-...',
});
```

## 预定义的 AI 厂商信息

### OpenAI

- 默认模型：gpt-4o-mini
- API URL: https://api.openai.com/v1
- 支持模型：gpt-4o, gpt-4o-mini, gpt-4-turbo, gpt-3.5-turbo
- 支持流式输出：✅
- 支持视觉功能：✅

### Anthropic (Claude)

- 默认模型：claude-3-5-sonnet-20241022
- API URL: https://api.anthropic.com
- 支持模型：claude-3-5-sonnet, claude-3-opus, claude-3-haiku
- 支持流式输出：✅
- 支持视觉功能：✅

### Kimi (月之暗面)

- 默认模型：moonshot-v1-8k
- API URL: https://api.moonshot.cn/v1
- 支持模型：kimi-k2, moonshot-v1-8k/32k/128k
- 支持流式输出：✅
- 支持视觉功能：❌

### 智谱 GLM

- 默认模型：glm-4-flash
- API URL: https://open.bigmodel.cn/api/paas/v4
- 支持模型：glm-4-plus, glm-4-air, glm-4-flash
- 支持流式输出：✅
- 支持视觉功能：✅
