## 使用方法

### 访问设置页面

1. **方法一**：点击右上角的 Settings 图标
2. **方法二**：直接在导航栏切换到设置标签（如果添加到导航）

### 切换设置模块

设置页面包含 4 个标签页，通过顶部导航切换：

```
┌─────────────────────────────────────────────┐
│ [🔑 AI 配置] [🎨 通用] [🔔 通知] [💾 存储] │
└─────────────────────────────────────────────┘
```

### AI 配置管理

在 AI 配置标签页中：

1. 查看已配置的 AI 厂商列表
2. 点击"添加"按钮新增厂商配置
3. 点击"编辑"按钮修改现有配置
4. 点击"删除"按钮移除配置
5. 使用"验证密钥"按钮测试 API 密钥有效性

## UI/UX 设计特点

### 1. 清晰的视觉层次

```
页面标题
├── 图标 + 文字说明
└── 副标题描述

标签导航
├── 4 个功能模块入口
└── 当前选中高亮

卡片布局
├── CardHeader（标题 + 描述）
├── CardContent（具体内容）
└── 统一的圆角和阴影
```

### 2. 一致的交互模式

- **卡片式布局**：所有功能模块使用统一的卡片样式
- **图标辅助**：每个标题都配有对应的图标
- **开关控件**：所有布尔值设置使用 Switch 组件
- **按钮分级**：普通操作使用 outline，危险操作使用 destructive

### 3. 友好的提示信息

- **功能描述**：每个模块都有详细的说明文字
- **安全提示**：AI 配置页面专门的安全教育
- **状态反馈**：存储空间使用进度条实时展示
- **警告标识**：危险操作使用红色警告区域

### 4. 响应式设计

- 使用 Tailwind CSS 的响应式类名
- 支持移动端和桌面端自适应
- 卡片宽度根据屏幕自动调整
