### 对于开发者 (安装Rust后)

#### 1. 安装Rust环境 (5-10分钟)

```powershell
# 以管理员身份运行 PowerShell
winget install Rustlang.Rustup
winget install Microsoft.EdgeWebView2Runtime

# 重启终端
# 验证安装
rustc --version  # 应 >= 1.70.0
```

#### 2. 启动应用 (1-2分钟)

```powershell
# 方法1: 使用一键脚本
.\scripts\setup-and-run.ps1 -Dev

# 方法2: 手动启动
npm install
npm run tauri:dev
```

## 📞 需要帮助?

### 常见问题

**Q: 如何验证Rust是否安装成功?**
```powershell
rustc --version
cargo --version
```

**Q: 启动应用需要多长时间?**
- 首次编译: 5-15分钟 (需要下载Rust依赖)
- 后续启动: 10-30秒 (使用缓存)

**Q: 可以在没有Rust的情况下开发吗?**
可以! 可以先用 `npm run dev` 开发前端部分，前端功能完整后再集成Rust后端。

**Q: 支持哪些操作系统?**
- Windows 10/11
- macOS 11+
- Linux (Ubuntu/Debian/Fedora等)
