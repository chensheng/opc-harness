# OPC-HARNESS 架构设计

> **文档版本**: v2.0  
> **最后更新**: 2026年3月21日  
> **架构类型**: 混合架构（云端AI + 本地工具）  
> **运行环境**: 用户本地计算机 + 云端AI服务

---

## 1. 架构概述

### 1.1 架构定位

OPC-HARNESS 采用**混合架构 (Hybrid Architecture)** 设计，作为一款本地桌面应用运行在用户计算机上，核心数据存储在本地，但AI能力通过对接多种云端AI厂商API实现，同时深度集成本地AI编码工具CLI，提供完整的Vibe Coding体验。

### 1.2 核心原则

| 原则 | 说明 | 实现方式 |
|------|------|---------|
| **多云AI支持** | 支持多家AI厂商，用户自主选择 | 统一AI适配层 + 多厂商API对接 |
| **本地优先存储** | 项目数据存储在本地文件系统 | SQLite + 本地文件存储 |
| **工具链集成** | 深度集成本地AI编码工具CLI | 进程调用 + CLI协议适配 |
| **密钥安全** | API密钥本地加密存储 | OS Keychain/Keystore |
| **离线降级** | 网络异常时有降级方案 | 本地缓存 + 离线提示 |

#### 4.1.2 Rust后端实现

```rust
// Rust 后端实现

use async_trait::async_trait;
use serde::{Deserialize, Serialize};

#[async_trait]
pub trait AIProvider: Send + Sync {
    fn name(&self) -> &str;
    fn supported_models(&self) -> Vec<ModelInfo>;
    
    async fn chat(&self, request: ChatRequest) -> Result<ChatResponse, AIError>;
    async fn stream_chat(
        &self, 
        request: ChatRequest,
        sender: tokio::sync::mpsc::Sender<String>
    ) -> Result<(), AIError>;
    
    async fn validate_key(&self, api_key: &str) -> Result<bool, AIError>;
}

// 厂商适配器实现
pub struct OpenAIAdapter {
    api_key: String,
    base_url: String,
    client: reqwest::Client,
}

pub struct KimiAdapter {
    api_key: String,
    client: reqwest::Client,
}

pub struct ClaudeAdapter {
    api_key: String,
    client: reqwest::Client,
}

pub struct GLMAdapter {
    api_key: String,
    client: reqwest::Client,
}

pub struct MiniMaxAdapter {
    api_key: String,
    client: reqwest::Client,
}

// AI服务管理器
pub struct AIServiceManager {
    providers: HashMap<String, Box<dyn AIProvider>>,
    default_provider: String,
    fallback_chain: Vec<String>, // 故障转移链
}

impl AIServiceManager {
    pub fn new() -> Self {
        let mut providers: HashMap<String, Box<dyn AIProvider>> = HashMap::new();
        
        // 注册各厂商适配器
        providers.insert("openai".to_string(), Box::new(OpenAIAdapter::new()));
        providers.insert("anthropic".to_string(), Box::new(ClaudeAdapter::new()));
        providers.insert("kimi".to_string(), Box::new(KimiAdapter::new()));
        providers.insert("glm".to_string(), Box::new(GLMAdapter::new()));
        providers.insert("minimax".to_string(), Box::new(MiniMaxAdapter::new()));
        
        Self {
            providers,
            default_provider: "openai".to_string(),
            fallback_chain: vec!["openai", "anthropic", "kimi"],
        }
    }
    
    pub async fn chat_with_fallback(&self, request: ChatRequest) -> Result<ChatResponse, AIError> {
        // 首选厂商
        if let Some(provider) = self.providers.get(&self.default_provider) {
            match provider.chat(request.clone()).await {
                Ok(response) => return Ok(response),
                Err(e) if e.is_retryable() => {
                    // 故障转移
                    for provider_name in &self.fallback_chain {
                        if let Some(fallback) = self.providers.get(provider_name) {
                            if let Ok(response) = fallback.chat(request.clone()).await {
                                return Ok(response);
                            }
                        }
                    }
                }
                Err(e) => return Err(e),
            }
        }
        Err(AIError::NoAvailableProvider)
    }
}
```

#### 4.2.2 CLI会话管理 (Rust)

```rust
use tokio::process::{Child, Command};
use tokio::io::{AsyncBufReadExt, AsyncWriteExt, BufReader};
use tokio::sync::mpsc;

pub struct CLISession {
    id: String,
    tool_type: String,
    process: Child,
    stdin: tokio::process::ChildStdin,
    stdout_rx: mpsc::Receiver<String>,
    stderr_rx: mpsc::Receiver<String>,
    working_dir: PathBuf,
}

pub struct CLIToolManager {
    sessions: Arc<Mutex<HashMap<String, CLISession>>>,
}

impl CLIToolManager {
    pub async fn create_session(
        &self,
        tool_type: &str,
        project_path: &Path,
    ) -> Result<String, CLIError> {
        let config = CLI_TOOLS.get(tool_type)
            .ok_or(CLIError::UnknownTool)?;
        
        // 检查工具是否安装
        if !self.check_tool_installed(&config.command).await? {
            return Err(CLIError::ToolNotInstalled(config.name.clone()));
        }
        
        let session_id = Uuid::new_v4().to_string();
        
        // 启动CLI进程
        let mut cmd = Command::new(&config.command);
        cmd.current_dir(project_path)
           .stdin(Stdio::piped())
           .stdout(Stdio::piped())
           .stderr(Stdio::piped());
        
        // 根据工具类型添加特定参数
        match tool_type {
            "kimi" => {
                cmd.arg("--session").arg(&session_id);
            }
            "claude" => {
                cmd.arg("--verbose");
            }
            "codex" => {
                cmd.arg("--model").arg("gpt-4o");
            }
            _ => {}
        }
        
        let mut process = cmd.spawn()?;
        
        let stdin = process.stdin.take().unwrap();
        let stdout = process.stdout.take().unwrap();
        let stderr = process.stderr.take().unwrap();
        
        // 创建输出通道
        let (stdout_tx, stdout_rx) = mpsc::channel(100);
        let (stderr_tx, stderr_rx) = mpsc::channel(100);
        
        // 启动输出读取任务
        self.spawn_output_reader(stdout, stdout_tx);
        self.spawn_output_reader(stderr, stderr_tx);
        
        let session = CLISession {
            id: session_id.clone(),
            tool_type: tool_type.to_string(),
            process,
            stdin,
            stdout_rx,
            stderr_rx,
            working_dir: project_path.to_path_buf(),
        };
        
        self.sessions.lock().await.insert(session_id.clone(), session);
        
        Ok(session_id)
    }
    
    pub async fn send_prompt(
        &self,
        session_id: &str,
        prompt: &str,
    ) -> Result<(), CLIError> {
        let mut sessions = self.sessions.lock().await;
        let session = sessions.get_mut(session_id)
            .ok_or(CLIError::SessionNotFound)?;
        
        // 根据工具协议格式化输入
        let formatted_prompt = match session.tool_type.as_str() {
            "kimi" => format!("{}\n", prompt),
            "claude" => format!("{}\n", prompt),
            "codex" => format!("{}\n", prompt),
            _ => format!("{}\n", prompt),
        };
        
        session.stdin.write_all(formatted_prompt.as_bytes()).await?;
        session.stdin.flush().await?;
        
        Ok(())
    }
    
    pub async fn read_output(
        &self,
        session_id: &str,
    ) -> Result<(Option<String>, Option<String>), CLIError> {
        let mut sessions = self.sessions.lock().await;
        let session = sessions.get_mut(session_id)
            .ok_or(CLIError::SessionNotFound)?;
        
        // 非阻塞读取
        let stdout = session.stdout_rx.try_recv().ok();
        let stderr = session.stderr_rx.try_recv().ok();
        
        Ok((stdout, stderr))
    }
    
    fn spawn_output_reader<R: AsyncBufReadExt + Unpin + Send + 'static>(
        &self,
        reader: R,
        tx: mpsc::Sender<String>,
    ) {
        tokio::spawn(async move {
            let mut reader = BufReader::new(reader);
            let mut line = String::new();
            
            loop {
                line.clear();
                match reader.read_line(&mut line).await {
                    Ok(0) => break, // EOF
                    Ok(_) => {
                        if tx.send(line.trim().to_string()).await.is_err() {
                            break;
                        }
                    }
                    Err(_) => break,
                }
            }
        });
    }
}
```

```typescript
// React组件: CLI控制台

import React, { useEffect, useRef, useState } from 'react';
import { invoke } from '@tauri-apps/api/core';

interface CLITerminalProps {
  sessionId: string;
  toolType: string;
}

export const CLITerminal: React.FC<CLITerminalProps> = ({ sessionId, toolType }) => {
  const [output, setOutput] = useState<string[]>([]);
  const [input, setInput] = useState('');
  const [isRunning, setIsRunning] = useState(true);
  const outputEndRef = useRef<HTMLDivElement>(null);
  const inputRef = useRef<HTMLInputElement>(null);

  // 轮询读取CLI输出
  useEffect(() => {
    const interval = setInterval(async () => {
      try {
        const [stdout, stderr] = await invoke<[string | null, string | null]>(
          'read_cli_output',
          { sessionId }
        );
        
        if (stdout) {
          setOutput(prev => [...prev, { type: 'stdout', content: stdout }]);
        }
        if (stderr) {
          setOutput(prev => [...prev, { type: 'stderr', content: stderr }]);
        }
      } catch (error) {
        console.error('Failed to read CLI output:', error);
      }
    }, 100);

    return () => clearInterval(interval);
  }, [sessionId]);

  // 自动滚动到底部
  useEffect(() => {
    outputEndRef.current?.scrollIntoView({ behavior: 'smooth' });
  }, [output]);

  const handleSend = async () => {
    if (!input.trim()) return;
    
    setOutput(prev => [...prev, { type: 'input', content: input }]);
    
    try {
      await invoke('send_cli_prompt', {
        sessionId,
        prompt: input,
      });
      setInput('');
    } catch (error) {
      setOutput(prev => [...prev, { type: 'error', content: String(error) }]);
    }
  };

  const handleStop = async () => {
    await invoke('stop_cli_session', { sessionId });
    setIsRunning(false);
  };

  return (
    <div className="flex flex-col h-full bg-slate-900 rounded-lg overflow-hidden">
      {/* 工具栏 */}
      <div className="flex items-center justify-between px-4 py-2 bg-slate-800 border-b border-slate-700">
        <div className="flex items-center gap-2">
          <span className="text-sm font-medium text-slate-200">
            {CLI_TOOLS[toolType]?.name || toolType}
          </span>
          <span className={`w-2 h-2 rounded-full ${isRunning ? 'bg-green-500' : 'bg-red-500'}`} />
        </div>
        <button
          onClick={handleStop}
          disabled={!isRunning}
          className="px-3 py-1 text-xs bg-red-600 hover:bg-red-700 text-white rounded disabled:opacity-50"
        >
          停止
        </button>
      </div>

      {/* 输出区域 */}
      <div className="flex-1 overflow-y-auto p-4 font-mono text-sm">
        {output.map((line, index) => (
          <div
            key={index}
            className={`mb-1 ${
              line.type === 'stderr' ? 'text-red-400' :
              line.type === 'input' ? 'text-blue-400' :
              line.type === 'error' ? 'text-red-500 font-bold' :
              'text-slate-300'
            }`}
          >
            {line.type === 'input' && <span className="text-slate-500 mr-2">&gt;</span>}
            {line.content}
          </div>
        ))}
        <div ref={outputEndRef} />
      </div>

      {/* 输入区域 */}
      <div className="flex items-center gap-2 p-4 border-t border-slate-700">
        <span className="text-slate-500">&gt;</span>
        <input
          ref={inputRef}
          type="text"
          value={input}
          onChange={(e) => setInput(e.target.value)}
          onKeyDown={(e) => e.key === 'Enter' && handleSend()}
          disabled={!isRunning}
          placeholder="输入指令或询问AI..."
          className="flex-1 bg-transparent text-slate-200 outline-none"
        />
        <button
          onClick={handleSend}
          disabled={!isRunning || !input.trim()}
          className="px-4 py-2 bg-blue-600 hover:bg-blue-700 text-white rounded disabled:opacity-50"
        >
          发送
        </button>
      </div>
    </div>
  );
};
```

```typescript
// 工具检测服务

interface ToolStatus {
  name: string;
  installed: boolean;
  version?: string;
  path?: string;
  installUrl: string;
  isRequired: boolean;
}

export const ToolDetectionService = {
  async detectAllTools(): Promise<ToolStatus[]> {
    const tools = [
      { name: 'Node.js', command: 'node', required: true },
      { name: 'Git', command: 'git', required: true },
      { name: 'Kimi CLI', command: 'kimi', required: false },
      { name: 'Claude Code', command: 'claude', required: false },
      { name: 'Codex CLI', command: 'codex', required: false },
      { name: 'Docker', command: 'docker', required: false },
    ];

    return Promise.all(
      tools.map(async (tool) => {
        try {
          const result = await invoke<{ installed: boolean; version?: string; path?: string }>(
            'detect_tool',
            { command: tool.command }
          );
          return {
            name: tool.name,
            installed: result.installed,
            version: result.version,
            path: result.path,
            installUrl: this.getInstallUrl(tool.name),
            isRequired: tool.required,
          };
        } catch {
          return {
            name: tool.name,
            installed: false,
            installUrl: this.getInstallUrl(tool.name),
            isRequired: tool.required,
          };
        }
      })
    );
  },

  getInstallUrl(toolName: string): string {
    const urls: Record<string, string> = {
      'Node.js': 'https://nodejs.org/',
      'Git': 'https://git-scm.com/downloads',
      'Kimi CLI': 'https://www.moonshot.cn/docs/cli',
      'Claude Code': 'https://docs.anthropic.com/en/docs/agents-and-tools/claude-code',
      'Codex CLI': 'https://github.com/openai/codex',
      'Docker': 'https://www.docker.com/products/docker-desktop',
    };
    return urls[toolName] || '#';
  },
};
```
