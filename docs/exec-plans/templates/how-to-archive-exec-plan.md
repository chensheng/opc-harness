# 执行计划归档指引

**描述**: 必须严格按照此指引归档执行计划，以便后续的任务跟踪和评估。

**步骤**:

## 1. 移动文件

```bash
mv docs/exec-plans/active/{TASK_ID}-{TASK_NAME}.md \
   docs/exec-plans/completed/{TASK_ID}-{TASK_NAME}.md
```

## 2. 更新文档状态

```
# {TASK_ID}: {TASK_NAME}

**状态**: ✅ 已完成
**完成日期**: YYYY-MM-DD
**总耗时**: X 天
**负责人**: OPC-HARNESS Team
```

```

**文档路径**: `path/to/doc.md`
```
