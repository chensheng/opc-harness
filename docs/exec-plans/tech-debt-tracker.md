## 🟡 中等技术债务 (P1-P2)

### TD-001: Rust Option<String>类型转换问题

- **创建日期**: 2026-03-16
- **优先级**: P2
- **状态**: 📋 待解决
- **影响范围**: `src-tauri/src/db/mod.rs`
- **描述**: 
  - `create_project`和 `update_project` 函数中，`Option<String>` 类型与 rusqlite 参数绑定不匹配
  - 当前使用 `.as_str()` 导致编译错误
  
- **根本原因**:
  - rusqlite 需要 `&str` 类型参数
  - `Option<String>.map_or()` 返回类型推断错误
  
- **解决方案**:
  ```rust
  // ❌ 当前错误代码
  .bind(2, project.description.map_or("", |d| d.as_str()))?
  
  // ✅ 修复方案：使用 as_deref()
  .bind(2, project.description.as_deref())?
  ```
  
- **偿还计划**: 2026-03-24 周
- **负责人**: 未分配

### 自动化清理

运行以下命令自动清理轻微技术债务:

```bash

npm run harness:fix:dry  # 预览

# 清理构建产物

rm -rf dist/
```
