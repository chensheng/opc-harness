## 📐 架构设计

### 技术栈
- **后端**: Rust + Tauri Commands
- **前端**: TypeScript + Vue 3
- **AI Provider**: OpenAI/Anthropic/Kimi/GLM
- **路由策略**: Quality-first（质量优先）

### 模块结构
```
src-tauri/src/
├── commands/
│   └── ai.rs          # Tauri Commands (新增 generate_prd 真实实现)
├── ai/
│   ├── mod.rs         # AI Provider 基础
│   └── router/
│       └── mod.rs     # 智能路由器 (已实现)
└── prompts/
    └── prd_template.rs # PRD 提示词模板 (需增强)
```

### 数据流
```
用户输入想法 
  ↓
Tauri Command `generate_prd`
  ↓
智能路由系统 → 选择最优 AI Provider
  ↓
构建 Prompt → 调用 AI API
  ↓
流式响应处理 → SSE 推送
  ↓
PRD 结构化解析 → 存入数据库
  ↓
前端实时展示
```
