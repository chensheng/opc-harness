## 🎯 目标

基于 MVP 版本完成的 UI 组件和架构，实现所有 mock 数据的真实 AI API 接入，让 OPC-HARNESS 真正具备 AI 驱动的产品设计、编码和营销能力。

### 核心理念

- **"人类掌舵，Agent 执行"** - 从 mock 到真实 AI 执行
- **渐进式披露** - 保持现有的分层信息展示
- **HITL (Human-in-the-Loop)** - 关键决策点人工介入

### 核心功能范围

1. **AI 适配器完整实现** - OpenAI/Claude/Kimi/GLM 真实 API 调用
2. **Vibe Design 真实化** - PRD/用户画像/竞品分析真实 AI 生成
3. **Vibe Coding Agent 执行** - Initializer/Coding Agent 真实运行
4. **Vibe Marketing 真实化** - 发布策略/营销文案真实 AI 生成
5. **WebSocket 实时通信** - 前端与后端 Agent 实时连接
6. **数据库持久化** - SQLite 存储项目数据和 AI 会话记录
