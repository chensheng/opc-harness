快速验证指令 - 直接复制到 Kimi CLI 使用
========================================================

【基础验证】
请帮我验证 http://localhost:1420：
1. 页面是否正常加载，标题是否包含 "OPC-HARNESS"
2. 页面是否有明显的错误提示
3. 主要内容区域是否可见

【导航验证】
请帮我验证 http://localhost:1420 的导航功能：
1. 查看导航菜单，确认包含 Dashboard、Idea、Coding、Marketing
2. 点击 "Idea" 菜单，检查是否能进入 Idea 输入页面
3. 返回首页，检查 Coding 工作区是否可访问

【表单验证】
请帮我验证 http://localhost:1420/idea 页面：
1. 页面是否正常加载
2. 是否有文本输入区域（textarea 或 input）
3. 是否有提交或生成按钮
4. 尝试在输入框中输入文字，确认可以输入

【完整验证】
请帮我执行完整的冒烟测试：

目标：http://localhost:1420

步骤：
1. 访问首页，确认标题为 "OPC-HARNESS"
2. 检查导航菜单是否包含：Dashboard、Idea、Coding、Marketing、Settings
3. 点击 Idea 导航，确认进入 Idea 输入页面（有输入框和按钮）
4. 返回 Dashboard，检查 Coding 工作区是否可访问
5. 检查 AI 配置页面是否可访问
6. 检查页面是否有 JavaScript 错误（控制台无红色错误）

请报告每个步骤的通过情况。

【性能检查】
请帮我检查 http://localhost:1420 的性能：
1. 页面加载时间是否在合理范围内（< 10秒）
2. 是否有资源加载失败（404错误）
3. 页面是否有卡顿或明显的性能问题

【响应式检查】
请帮我验证 http://localhost:1420 的布局：
1. 检查是否有水平滚动条（应该有）
2. 元素是否有重叠
3. 文字是否可读，是否有截断
4. 按钮是否可见且可点击

使用说明：
1. 先启动开发环境：npm run tauri:dev
2. 复制上面的指令粘贴到 Kimi CLI
3. 或使用：@browser http://localhost:1420
4. 然后根据看到的内容回答验证问题
